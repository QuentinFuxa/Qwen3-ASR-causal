# Qwen3-ASR causal

Standalone runtime for a causal streaming Qwen3-ASR audio tower.

The model weights live on Hugging Face:
[qfuxa/qwen3-asr-0.6b-streaming](https://huggingface.co/qfuxa/qwen3-asr-0.6b-streaming).
This repository contains the runtime and examples; it does not commit model
weights.
